# Descrição do Trabalho
Este trabalho apresenta o desenvolvimento de uma aplicação em Java para o cálculo da força de atrito e análise do movimento de um bloco em uma superfície horizontal rugosa. O sistema foi desenvolvido com o objetivo de aplicar conceitos de Física e Programação, realizando cálculos como força normal, força de atrito, força resultante e aceleração.

O programa permite ao usuário inserir dados como massa, força aplicada, ângulo, coeficiente de atrito e gravidade, fornecendo como resultado a situação do bloco, indicando se ele permanece em repouso ou entra em movimento. Foram desenvolvidas versões com interface gráfica e em modo console, permitindo execução em diferentes ambientes.
